Upgrading

To upgrade FlareLint on your computer, you need to uninstall the current Python FlareLint package then install your new Python FlareLint package.

This task does not affect your FlareLint rules. The upgraded FlareLint tool will use your existing rules. Also, this task does not affect your Python installation. FlareLint continues to use Python.

To uninstall FlareLint:

  1. Download and extract the contents of the FlareLint.zip installer for the new FlareLint tool.

    www.intelerad.com/flarelint

    Tip: Unzip FlareLint.zip into a folder on your desktop.

  2. Open the Uninstall or Change a Program control panel.

    In the Windows Start menu, click Control Panel. Under Programs, click Uninstall a Program.

  3. Double-click Python FlareLint-x, where x is the version of the currently-installed FlareLint.

  4. In the Programs and Features alert, click Yes to confirm that you are sure you want to uninstall the old version of FlareLint.

  5. In the unzipped folder, double-click the FlareLint.msi file.

  6. In the Python FlareLint Setup wizard, enable Install Just for Me and click Next.

  7. Choose Python x from Registry and click Next.

    The installation begins.

  8. If the User Account Control alert appears to prompt you to allow the installer to change your computer, click Yes.

  9. When the wizard is done, click Finish.
