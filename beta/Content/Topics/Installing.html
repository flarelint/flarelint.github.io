Installing

Before using FlareLint for the first time, you need to install the Python programming language and the FlareLint package.

There are many variations of the Python installer to handle Python versions, host operating systems, and installation methods. But FlareLint requires the Python installer that meets these requirements:

  • Python version 3 or later
  • latest release version, not an alpha or beta version, which have "a" or "b" at the end of the version number
  • the executable installer for Microsoft Windows, x86-64-bit

To install Python:

  1. Find and download the correct installer for Python 3:

    www.python.org/downloads

    Tip: The latest Python 3 release executable installer for Microsoft Windows 64-bit is the following: www.python.org/ftp/python/3.6.0/python-3.6.0-amd64.exe.

  2. Double-click the installer for Python 3 for 64-bit Windows.

  3. If Windows displays the Open File - Security Warning alert, click Run.

  4. In the Python Setup wizard, enable the following options:

    • Install Launcher for All Users.
    • Add Python x to PATH, where x is the version of Python you are installing.

  5. Click Install Now

  6. If Windows displays the User Account Control alert, click Yes to continue.

    The installation starts.

  7. When the installer finishes, click Close.

To install FlareLint:

  1. Install Python.

  2. If you have not done so already, extract the contents of the FlareLint.zip installer.

    Tip: Unzip FlareLint.zip into a folder on your desktop.

  3. In the unzipped folder, double-click the FlareLint.msi file.

  4. In the Python FlareLint Setup wizard, enable Install Just for Me and click Next.

  5. Choose Python x from Registry, where x is the version of Python that you installed, and click Next.

    The installation begins.

  6. If the User Account Control alert appears to prompt you to allow the installer to change your computer, click Yes.

  7. When the wizard is done, click Finish.

  8. Copy FlareLint.bat and paste it into a folder that you can access easily.

    For example, paste it onto your desktop.

  9. Configure your personal rules directory, as needed. See Managing Rules.
