pov-attribute-contains

pov-attribute-contains tag name "text"
pov-attribute-contains * name "text"

True when an element has an attribute name and its value contains text. False when either an element does not have an attribute name or, if it does have this attribute, its value does not contain text. This predicate is case-sensitive.

The point of view, pov may be one of the following: after, ancestor, before, child, descendant, next, parent, previous, and self.

Use the wildcard (“*”) to match any element tag.

For example:

<a href="#Fredericton">Home of Bliss Carman</a>

For the a element, the following give true:

self-attribute-contains a href "Fred"
self-attribute-contains * href "deric"

And the following give false:

self-attribute-contains * href "John"
self-attribute-contains a href "#fred"
