pov-attribute-equals

pov-attribute-equals tag name "text"
pov-attribute-equals * name "text"

True when an element has an attribute name and its value is text. False when either an element does not have an attribute name or, if it does have this attribute, its value is not text.

Use the wildcard (“*”) to match any element tag.

The point of view, pov may be one of the following: after, ancestor, before, child, descendant, next, parent, previous, and self.

This predicate is case-sensitive. Leading and ending spaces and line endings are not ignored.

For example:

<a href=" #Fredericton">Home of Bliss Carman</a>

For the a element, the following give true:

self-attribute-equals a href " #Fredericton"
self-attribute-equals * href " #Fredericton"

And the following give false:

self-attribute-equals a href "#Fredericton "
self-attribute-equals * href " #fredericton"
