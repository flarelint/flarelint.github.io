pov-attribute

pov-attribute tag name
pov-attribute * name

True when an element tag has a name attribute with a value that is not empty, false otherwise.

Use the wildcard (“*”) to match any element tag.

The point of view, pov may be one of the following: after, ancestor, before, child, descendant, next, parent, previous, and self.

For example:

<p style="color: blue;">Pelee Island</p>

the predicate self-attribute p style gives true and self-attribute h1 style gives false.

In the following:

<h1 style="">Wolf Island</h1>

the predicate self-attribute p style gives false, self-attribute h1 style gives false, and self-attribute * style gives false.

In the following:

<li>Manitoulin Island</li>

the predicate self-attribute li style gives false and self-attribute * style gives false.

Spaces and line breaks at the beginning and end of the attribute value are ignored.

For example:

<a href="   " title="   Population 30">Dorval Island</a>

For the a element, the following gives false:

self-attribute a href

And the following gives true:

self-attribute a title
