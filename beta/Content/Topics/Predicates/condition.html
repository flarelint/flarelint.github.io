condition

condition set.name

Is true when an element has a condition of set.name or is in an element that has this condition, false otherwise.

The condition must use the format of condition set, period (.), then a name defined in the condition set.

For example:

<h1>New Brunswick Bucket List</h1>
<p>Never negate the boundless beauty of these places:</p>
<ul MadCap:conditions="Acme.ScreenOnly">
  <li>Edmunston</li>
  <li>Saint John</li>
  <li>Kouchibouguac</li>
</ul>

For the ul element and the li elements, the following predicate gives true:

condition Acme.ScreenOnly
