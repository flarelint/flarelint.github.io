snippet "snippet-file"

True when an element is a link to a snippet, snippet-file, false otherwise. This predicate is case-sensitive.

Flare has two kinds of inserted snippets, text and block. A text snippet is in-line with other content. For example. a paragraph that contains a snippet would be encoded as a link to a text snippet. A block snippet occupies an entire blank line in a topic body.

For example:

<body>
  <h1>Report on Executive Decisions</h1>
  <p>Lord Ronald ran madly off in all directions.</p>
  <MadCap:snippetBlock src="Resources/Snippets/AcmeDisclaimer.flsnp">
</body>

For the MadCap:snippetBlock element, the following gives true:

snippet "Resources/Snippets/AcmeDisclaimer.flsnp"
