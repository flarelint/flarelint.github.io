pov-style

pov-style tag
pov-style tag.class
pov-style .class

True when the element's tag and its class attribute are tag and class, respectively. This predicate uses a similar format to the Style Picker and Styles window in Flare. In these windows, the name after a period (.) is the class attribute in the element's XML.

If tag is not specified, then it is ignored. If class is not specified then it matches only an empty class attribute.

The point of view, pov may be one of the following: after, ancestor, before, child, descendant, next, parent, previous, and self.

For example, when scanning this element

<p class="Note">...</p>

these predicates each give a true result:

self-style p.Note
self-style .Note

and this predicate gives false:

self-style p

Alternately, with this element:

<p>...</p>

this predicate gives false:

style p.Note

and this predicate gives true:

style p
