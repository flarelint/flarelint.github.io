FlareLint and MadCap Analyzer

FlareLint complements MadCap Flare reports and MadCap Analyzer. MadCap Flare reports and MadCap Analyzer gather and report information about the state of a Flare project. For example, you can use these features to catch broken links and unused images.

On the other hand, FlareLint analyzes your content for adherence to your team's organizational style, writing conventions, and production requirements. Examples: 

  • Use FlareLint to make sure that your topics always start with an h1 element.
  • If you publish your targets on a Linux web server, add a rule to make sure that file names in your Flare project do not contain illegal characters for Linux file systems.
